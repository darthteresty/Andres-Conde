My Work at Ubisoft Bucharest

2021 - Feb 2025

As Game Designer in "Ubisoft Bucharest" my main work was the design and creation of a new system to elaborate each of the new "Manhunt Seasons" (later known as "Manhunt Scouts"), one of the core parts of the Seasonal Updates for "Tom Clancy's: The Division 2", with main focus on the narrative side of the game.

During this time I worked with multiple teams: Level Design, UI, Tech... but my main collaboration was with the GPP team to create each of the new mechanics and systems for the "Manhunt" (UI, rewards, collectibles, new enemies...) and Dev Test team, to make sure everything we added or modify during our updates was correctly shown and ready for the players. I became one of the main contact point for any design-question related to the "Manhunt Scouts" as the feature owner.

Although the "Manhunt" was an already stablished system when I first started to work on it, we as a team had to "re-invent it" due to the new players (since the game was released in 2019) and projects needs, evolving the system to it's new state, starting to create and keep a new documentation and making sure everyone in the team followed the new game direction.

We kept an organized directory by using Confluence, creating new data bases for every new Season and pointing out each change needed for the next one (based on internal an player feedback). We iterated multiple ideas for the new "Scouts" by using Miro an other collaborative tools.

I also worked very closely with the Narrative and Localization team too, ensuring both story and gameplay were linked in a good way for different types of players, attending to the constantly evolving gameplay elements and keeping the narrative cohesive. I worked directly in OASIS and in Snowdrop (the game's engine) to add each of the new collectibles and narrative content (such as the NPCs dialogues).
